O que é HTML?

O HTML é uma linguagem de marcação utilizada para estruturar e organizar as informações contidas dentro de um site.

Ele é construído com as chamadas tags, que funcionam como a definição do tipo de conteúdo que será exibido. Abaixo estão as principais tags e seus usos.

Principais Tags HTML

  • Títulos (h1 e h2):

    Utilizamos a tag `h` com números de 1 a 6 para definir o texto como título, onde h1 é o mais importante e o h6 é o menos importante.

  • Parágrafos (p):

    Utilizamos a tag `p` para indicar um parágrafo de texto.

  • Listas (ul, ol, li):

    Utilizamos a tag `ul` para criar uma lista não ordenada e `ol` para uma lista ordenada. Dentro de cada lista, usamos `li` para os itens.

    Exemplo de lista ordenada:

    1. Primeiro item
    2. Segundo item
    3. Terceiro item
  • Links (a):

    Utilizamos a tag `a` com o atributo `href` para criar links. Veja um exemplo de um link para o Google.

  • Imagens (img):

    Utilizamos a tag `img` com o atributo `src` para inserir imagens. Veja um exemplo de uma imagem:

  • Tabelas (table, tr, td):

    Utilizamos a tag `table` para criar tabelas. Dentro dela, usamos `tr` para definir as linhas e `td` para definir as células.

    Exemplo de tabela:

    Nome Idade
    Ana 28
    João 34
